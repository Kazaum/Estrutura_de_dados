--------------------------Dia 1: Aprendendo sobre Vetores (Arrays)-----------------------------

Vetores (Arrays): Neste primeiro dia, mergulhei nos conceitos dos Vetores, ou Arrays. Compreendi como criar e usar arrays de uma maneira mais eficiente, permitindo alterações e consultas de forma mais consciente. Isso não apenas torna o código mais dinâmico, mas também mais reutilizável, evitando bagunça no código.

Manipulação de Arrays Dinâmicas: Aprendi a adicionar novos valores dinamicamente a essas arrays. Essa abordagem orientada me ajudou a manter meu código organizado e limpo, garantindo uma estrutura bem organizada.

Utilização de Métodos para Acesso aos Valores: Além disso, dei os primeiros passos na impressão de valores presentes nessas arrays através de métodos como get e toString. Isso abriu diversas possibilidades de utilização, facilitando a manipulação dos dados presentes nos arrays de maneira mais intuitiva.
